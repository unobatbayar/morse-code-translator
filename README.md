## How to run
Simply clone it, and type `java Main` on your terminal in the cloned directory to run.

    Welcome to Morse Code translator by Uno! (type ':wq' to exit)
    -> Start by entering text or morse:
 
## Input 
    aaron swartz
    july 4 1776
    .... . .-.. .-.. --- /  .-- --- .-. .-.. -..
    .--- ..- .-.. -.--/ ....-/ .---- --... --... -....
    ..- -. ---/ -... .- - -... .- -.-- .- .-.
